#  Belajar MY SQL


### Melihat semua database yang ada 
```
show databases
```

### Membuat database
- disarankan menggunakan huruf kecil semua
create database nama_database 
```
create database nama_database
```

### Menghapus database

```
drop database nama_database;
```


### Memilih database

```
use nama_database;
```
